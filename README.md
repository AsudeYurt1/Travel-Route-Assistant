## Travel Route Assistant
## Overview

This project is an AI-supported desktop application that provides travel route recommendations based on user interests (e.g., museums, historical places, parks) and visualizes suggested locations on an interactive map.

## Features

Interest-based place recommendations

AI-powered route suggestion

Interactive map visualization using Folium

Desktop user interface built with Tkinter
